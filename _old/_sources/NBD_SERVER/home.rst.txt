Home
=============

Home is meant as the user interface of the cluster where
simulations&results should be kept for further analisis on
scratch

The current space is:

    - Capacity: 50GB
    - Occupied : 50GB  
    - Available: 0 
    - Usage : 100%

Under **/home/<user_name>** each user has his own private folder

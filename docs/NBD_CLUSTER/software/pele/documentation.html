PELE docs

Compilation¶

You can find a README file explaining how to compile PELE on NBD cluster under /work/NBD_Utilities/PELE/PELE_Compilation/intel

Tutorial¶

You can find PELE tutorial under the folder /work/NBD_Utilities/PELE/PELE_Tutorials/ where you will find:

  • PELE_Method.docx: Document on how the method works
  • EXAMPLE_RUNS_PELE.docx : Document with an overview of the tutorial.
  • PELE++_CONTROL_FILE.docx: Document on how to set PELE parameters.
  • PELE++_EXAMPLES.tar.gz: All example simulations to run.
  • PELE_PCA: Tutorial on how to produce a PCA from several X-rays and plug that one into PELE
  • PELE_PLATFORM: Tutorial on how to launch pele from NBDServer using PELE’s Platform

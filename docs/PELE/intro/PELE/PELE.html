PELE¶

PELE is an heuristic MC simulation software build as a 3-part algorithm as explained here .

Simulation Requirements¶

To start a simulation you must have:

  • Prepared the receptor with OPLS atom typing. Check the Protein Preparation section.
  • Prepared a DataLocal folder with the parameters of your ligand. Check the Ligand Prepareation section.
  • Prepared a PELE Control File. Check the `Control File <>`__ section.

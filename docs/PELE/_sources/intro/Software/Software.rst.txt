##########
Software 
##########

PELE is an heuristic MC simulation software build as a 3-part algorithm as explained `here <../../theory/theory.html>`__ .

**********************************
Requirements
**********************************

To start a simulation you must have:

    - Comercial PELE installed. Send and e-mail to: daniel.soler@nostrumbiodiscovery.com
    - Prepared the receptor with OPLS atom typing. Check the `Protein Preparation <../SystemPreparation/SystemPreparation.html#receptor-preparation>`__  section.
    - Prepared a DataLocal folder with the parameters of your ligand. Check the `Ligand Preparation <../SystemPreparation/SystemPreparation.html#ligand-preparation>`__  section.
    - Prepared a PELE Control File. Check the `Control File <../Simulation/Simulation.html#Input>`__  section.

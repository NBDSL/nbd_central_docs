PELE¶

PELE Tutorial¶

You can find PELE tutorial under the folder /data/software/PELE/Tutorial/ where you will find:

  • PELE_Method.docx: Document on how the method works
  • EXAMPLE_RUNS_PELE.docx : Document with an overview of the tutorial.
  • PELE++_CONTROL_FILE.docx: Document on how to set PELE parameters.
  • PELE++_EXAMPLES.tar.gz: All example simulations to run.
  • PELE_PCA: Tutorial on how to produce a PCA from several X-rays and plug that one into PELE
